# TOOLOR — Статус разработки

**Версия:** 1.0.0 (билд 12)
**Дата:** 5 апреля 2026
**Платформа:** iOS (TestFlight)

## Работает
- Каталог 86 товаров TOOLOR с актуальными фото, ценами, размерами
- Авторизация по SMS-коду (OTP в БД, защита от перебора)
- Корзина, оформление заказа, самовывоз/доставка
- Программа лояльности: 4 уровня, кэшбэк 3–12%, QR-код
- Оплата через Finik (карта/QR)
- Промокоды, списание баллов при покупке
- Админ-панель: загрузка каталога из Excel, управление заказами/пользователями, QR-сканер

## Не работает / в процессе
- SMS-провайдер (Nikita.kg) — ожидаем учётные данные
- Sentry мониторинг — ожидаем DSN
- Finik webhook на стороне Finik — ожидаем настройки у провайдера

## Свежий билд
TestFlight билд 12 — доступен через 5-10 минут после загрузки
